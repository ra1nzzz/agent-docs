### 2. P2P 文件传输工具

**功能:**
- Agent 之间直接文件传输
- 跨局域网/公网传输
- 小文件秒传，大文件分片
- 基于免费中继 API 做节点发现

**核心代码:**
```python
class SimpleP2P:
    def __init__(self, transfer_key: str):
        self.key = transfer_key
        self.relay_host = "p2p-relay.freeapis.cn"
        self.relay_port = 80
    
    def send_file(self, file_path: str) -> bool:
        """发送文件"""
        with open(file_path, "rb") as f:
            file_data = base64.b64encode(f.read()).decode()
        self._post_relay({
            "type": "file",
            "key": self.key,
            "data": file_data,
            "filename": file_path.split("/")[-1]
        })
        return True
    
    def receive_file(self, save_dir: str = "./") -> str:
        """接收文件"""
        file_data = self._listen_file()
        filename = file_data["filename"]
        save_path = f"{save_dir}/{filename}"
        with open(save_path, "wb") as f:
            f.write(base64.b64decode(file_data["data"]))
        return save_path
```

## 🔄 整合流程

### 1. Orchestration 整合流程

```
任务创建
    ↓
检查依赖关系
    ↓
构建 DAG (有向无环图)
    ↓
分配 Agent (考虑依赖)
    ↓
执行任务
    ↓
完成后触发依赖任务
    ↓
更新任务状态
```

### 2. P2P 文件传输流程

**发送方:**
```
选择文件
    ↓
Base64 编码
    ↓
上传到中继 API
    ↓
通知接收方 (密钥)
    ↓
等待接收确认
```

**接收方:**
```
接收密钥
    ↓
从中继 API 拉取
    ↓
Base64 解码
    ↓
保存到本地
    ↓
发送确认
```

### 3. 工作流执行流程

```
触发工作流 (定时/事件)
    ↓
执行节点 1
    ↓
检查条件
    ↓
执行节点 2
    ↓
...
    ↓
完成工作流
    ↓
记录执行历史
```
