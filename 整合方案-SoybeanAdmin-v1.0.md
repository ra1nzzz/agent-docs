## 🚀 部署步骤

### Phase 1: SoybeanAdmin 部署 (2 小时)

```bash
# 1. 克隆 SoybeanAdmin
cd /root
git clone https://github.com/soybeanjs/soybean-admin.git zhiyi-admin
cd zhiyi-admin

# 2. 安装依赖
npm install

# 3. 配置代理 (指向智弈 API)
cat > .env << EOF
VITE_API_BASE_URL=http://107.149.218.118:3008
VITE_WS_URL=ws://107.149.218.118:3008
EOF

# 4. 构建
npm run build

# 5. PM2 部署
pm2 start ecosystem.config.js
```
