## 🔄 飞书任务同步机制

### 同步方向

**双向同步:**
```
智弈集群任务 ←→ 飞书任务
     ↓              ↓
  SoybeanAdmin   飞书 App
```

### 同步策略

**1. 智弈 → 飞书:**
- 创建任务时勾选"同步到飞书"
- 任务状态变更时同步 (完成/暂停/恢复)
- 负责人变更时同步
- 截止时间变更时同步

**2. 飞书 → 智弈:**
- 每 5 分钟轮询飞书任务
- 检测任务状态变更
- 检测新增任务 (在默认清单中)
- 检测任务删除

### 数据映射

| 智弈字段 | 飞书字段 | 说明 |
|---------|---------|------|
| `task.title` | `task.summary` | 任务标题 |
| `task.description` | `task.description` | 任务描述 |
| `task.assignee` | `task.members[].id` | 负责人 (open_id) |
| `task.status` | `task.completed_at` | 状态映射 |
| `task.due_date` | `task.due.timestamp` | 截止时间 |
| `task.priority` | `task.custom_fields` | 优先级 (自定义字段) |

### 状态映射

| 智弈状态 | 飞书状态 | 说明 |
|---------|---------|------|
| `pending` | `completed_at=null` | 待处理 |
| `working` | `completed_at=null` | 进行中 |
| `completed` | `completed_at!=null` | 已完成 |
| `paused` | `custom_fields.paused=true` | 已暂停 |

### 冲突解决

**最后写入优先 (Last Write Wins):**
- 记录每次修改的时间戳
- 同步时比较时间戳
- 较新的数据覆盖较旧的数据

**手动冲突解决:**
- 检测到冲突时弹窗提示
- 用户选择保留哪个版本
- 或手动合并
