## 项目概述

### 背景

智弈集群已建立多 Agent 协作系统（Lucy/小桔/小深），但缺少：
- ❌ 统一的可视化监控界面
- ❌ Agent 状态实时展示
- ❌ Token 用量统计与趋势
- ❌ 任务流转监督面板
- ❌ 团队运营数据汇总

### 目标

基于 [AgentTeamsBI](https://github.com/sunxi1993-a11y/AgentTeamsBI) 构建**智弈集群工作频道主界面**，实现：
- ✅ 一站式监控所有 Agent 状态
- ✅ Token 用量可视化（Gateway + 百炼）
- ✅ 任务全流程追踪
- ✅ 团队运营数据汇总
- ✅ 飞书集成推送

### 核心价值

| 受益方 | 价值 |
|--------|------|
| **韬哥** | 一眼掌握全局，无需多系统切换 |
| **Agent 团队** | 状态透明，工作量可量化 |
| **运营分析** | 数据驱动优化决策 |

### 数据流

```
1. Agent 心跳 → 智弈枢纽 → AgentTeamsBI (状态面板)
2. Gateway 日志 → 解析脚本 → AgentTeamsBI (用量面板)
3. 百炼 API → 监控脚本 → AgentTeamsBI (套餐面板)
4. 任务事件 → 智弈枢纽 → AgentTeamsBI (任务面板)
5. 关键指标 → 飞书机器人 → 韬哥 (推送通知)
```

### 4️⃣ 飞书集成推送

**推送规则：**

| 推送类型 | 频率 | 触发条件 | 内容 |
|----------|------|----------|------|
| 日报 | 每天 9:00 | 定时 | 昨日用量 + 任务汇总 |
| 异常报警 | 实时 | Agent 离线/任务失败 | 告警信息 |
| 周报 | 每周一 9:00 | 定时 | 上周工作量排行 |
| 用量预警 | 实时 | 百炼用量>80% | 余量提醒 |

**推送示例：**
```

  小桔   🟢 在线  今日任务：8✅ 5⏳
  小深   🟡 忙碌  今日任务：5✅ 2⏳

📈 Token 用量
  Gateway: 26,542 tokens
  百炼：23% (5h) / 9% (7d) / 46% (30d)
  套餐剩余：17 天

📋 任务汇总
  完成：25  进行中：10  失败：1
```

## 技术实现

### 技术栈

| 组件 | 技术 | 说明 |
|------|------|------|
| **前端** | React + TypeScript | 响应式界面 |
| **后端** | Python 3.x | FastAPI/Flask |
| **数据源** | 智弈 API + Gateway 日志 | REST API |
| **部署** | 本地运行 | localhost:7891 |
| **推送** | 飞书机器人 | Webhook |

### 目录结构

```
AgentTeamsBI/
├── dashboard/              # 后端服务
│   ├── server.py          # 主服务
│   ├── dist/              # 前端静态文件
│   ├── task_events.json   # 任务事件
│   ├── agents_status.json # Agent 状态
│   └── usage.json         # Token 用量
├── edict/                 # 前端源码
│   └── frontend/          # React 项目
├── integrations/          # 集成脚本
│   ├── zhiyi_sync.py      # 智弈数据同步
│   ├── gateway_parser.py  # Gateway 日志解析
│   └── bailian_sync.py    # 百炼用量同步
└── docs/                  # 文档
    └── 智弈集群工作频道方案.md
```

### API 接口

**智弈集群枢纽 API 增强：**

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/agents/status` | GET | Agent 状态列表 |
| `/api/agents/:id/heartbeat` | POST | 心跳上报 |
| `/api/tasks/list` | GET | 任务列表 |
| `/api/tasks/:id` | GET | 任务详情 |
| `/api/usage/token` | GET | Token 用量统计 |

**AgentTeamsBI API：**

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/dashboard/overview` | GET | 仪表盘总览 |
| `/api/dashboard/agents` | GET | Agent 状态 |
| `/api/dashboard/usage` | GET | Token 用量 |
| `/api/dashboard/tasks` | GET | 任务日志 |

## 风险与应对

### 技术风险

| 风险 | 概率 | 影响 | 应对措施 |
|------|------|------|----------|
| 智弈 API 不稳定 | 中 | 高 | 增加缓存机制 |
| Gateway 日志格式变化 | 低 | 中 | 正则兼容多版本 |
| 百炼 API 限流 | 中 | 中 | 降低查询频率 |
| 前端构建失败 | 低 | 低 | 使用预构建版本 |

### 运营风险

| 风险 | 概率 | 影响 | 应对措施 |
|------|------|------|----------|
| Agent 离线未感知 | 中 | 高 | 飞书实时报警 |
| Token 超量 | 低 | 高 | 80% 用量预警 |
| 任务堆积 | 中 | 中 | 任务队列监控 |
