## 项目概述

### 背景

智弈集群已建立多 Agent 协作系统（Lucy/小桔/小深），但缺少：
- ❌ 统一的可视化监控界面
- ❌ Agent 状态实时展示
- ❌ Token 用量统计与趋势
- ❌ 任务流转监督面板
- ❌ 团队运营数据汇总

### 目标

基于 [AgentTeamsBI](https://github.com/sunxi1993-a11y/AgentTeamsBI) 构建**智弈集群工作频道主界面**，实现：
- ✅ 一站式监控所有 Agent 状态
- ✅ Token 用量可视化（Gateway + 百炼）
- ✅ 任务全流程追踪
- ✅ 团队运营数据汇总
- ✅ 飞书集成推送

### 核心价值

| 受益方 | 价值 |
|--------|------|
| **韬哥** | 一眼掌握全局，无需多系统切换 |
| **Agent 团队** | 状态透明，工作量可量化 |
| **运营分析** | 数据驱动优化决策 |

3. **无数据库** - 仅 JSON 文件存储，易损坏
4. **配置硬编码** - 端口、路径等未使用环境变量
5. **无单元测试** - 代码质量无保障

## 系统架构

### 整体架构图

```
┌─────────────────────────────────────────────────────────────────┐
│                        智弈集群工作频道                          │
│                     http://localhost:7891                       │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐         │
│  │  Agent 状态   │  │  Token 用量   │  │  任务监督    │         │
│  │   监控面板    │  │   统计面板    │  │   日志面板    │         │
│  └──────────────┘  └──────────────┘  └──────────────┘         │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
                              ↑
                              │ 数据同步
        ┌─────────────────────┼─────────────────────┐
        ↓                     ↓                     ↓
┌───────────────┐   ┌───────────────┐   ┌───────────────┐
│  智弈集群枢纽  │   │ OpenClaw      │   │  百炼监控     │
│ agent.ytaiv.com│   │ Gateway 日志   │   │  套餐余量     │
└───────────────┘   └───────────────┘   └───────────────┘
        ↑
        │ 心跳上报
┌───────┴───────┬───────────────┐
│               │               │
┌───────┐  ┌───────┐  ┌───────┐
│ Lucy  │  │ 小桔  │  │ 小深  │
└───────┘  └───────┘  └───────┘

                              ↓
                    ┌───────────────┐
                    │   飞书推送    │
                    │  定时摘要     │
                    └───────────────┘
```

### 数据流

```
1. Agent 心跳 → 智弈枢纽 → AgentTeamsBI (状态面板)
2. Gateway 日志 → 解析脚本 → AgentTeamsBI (用量面板)
3. 百炼 API → 监控脚本 → AgentTeamsBI (套餐面板)
4. 任务事件 → 智弈枢纽 → AgentTeamsBI (任务面板)
5. 关键指标 → 飞书机器人 → 韬哥 (推送通知)
```
