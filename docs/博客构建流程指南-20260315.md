# 📝 博客构建流程指南

**项目:** 弈韬 AI 观察 (ytao-ai-observer)  
**域名:** https://ytaiv.com  
**框架:** Astro + vhAstro-Theme  
**部署:** VPS (HK) + Caddy 自动 HTTPS

---

## 🚀 快速开始

### 1. 克隆项目

```bash
cd ~/.openclaw/workspace
git clone https://github.com/ra1nzzz/ytao-ai-observer.git
cd ytao-ai-observer
```

### 2. 安装依赖

```bash
# 使用 pnpm（推荐）
pnpm install

# 或使用 npm
npm install
```

### 3. 创建新文章

```bash
# 文章模板位置
src/content/blog/

# 命名格式：YYYY-MM-DD-slug.md
# 示例：2026-03-15-himalaya-email-setup.md
```

### 4. 文章 Frontmatter

```markdown
---
title: "文章标题"
description: "文章描述（150 字以内）"
pubDate: 2026-03-15
tags: ["标签 1", "标签 2"]
author: "韬哥"
image: "/assets/images/blog/封面图.jpg"
draft: false  # true=草稿，false=发布
---

# 文章正文

内容...
```

## 🔨 构建流程

### 本地预览

```bash
# 启动开发服务器
pnpm dev

# 访问 http://localhost:4321
```

### 生产构建

```bash
# 清理旧构建
pnpm run clean

# 构建生产版本
pnpm run build

# 预览构建结果
pnpm run preview
```

### 部署到 VPS

```bash
# 方式 1: Git 推送（推荐）
git add .
git commit -m "发布：文章标题"
git push origin main

# VPS 自动部署（配置了 Git Hook）
```

```bash
# 方式 2: 手动部署
# 1. 本地构建
pnpm run build

# 2. 上传到 VPS
scp -r dist/* root@ytaiv.com:/var/www/ytao-ai-observer/

# 3. 重启 Caddy
ssh root@ytaiv.com "systemctl restart caddy"
```

## 🎨 图片处理

### 图片位置

```
public/assets/images/blog/
```

### Markdown 中引用

```markdown
![描述](/assets/images/blog/图片名.jpg)
```

### 优化建议

- 使用 WebP 格式（更小体积）
- 压缩图片（TinyPNG）
- 尺寸适中（宽度 1200px 以内）

## ⚠️ 常见问题

### 1. 构建失败

```bash
# 清理缓存
rm -rf node_modules .astro dist
pnpm install
pnpm run build
```

### 2. 图片不显示

```bash
# 检查图片路径
# 正确：/assets/images/blog/xxx.jpg
# 错误：./assets/images/blog/xxx.jpg

# 检查文件大小写
# Linux 区分大小写：Image.jpg ≠ image.jpg
```

### 3. 部署后 404

```bash
# 检查 Caddy 配置
ssh root@ytaiv.com
cat /etc/caddy/Caddyfile

# 重启 Caddy
systemctl restart caddy
```

## 🔧 自动化脚本

### 发布新文章脚本

```bash
#!/bin/bash
# publish.sh

TITLE="$1"
DATE=$(date +%Y-%m-%d)
SLUG=$(echo $TITLE | tr '[:upper:]' '[:lower:]' | tr ' ' '-')

# 创建文章
cat > src/content/blog/${DATE}-${SLUG}.md << EOF
---
title: "$TITLE"
description: "文章描述"
pubDate: $DATE
tags: ["AI", "技术"]
author: "韬哥"
draft: true
---

# $TITLE

内容...
EOF

echo "✅ 文章已创建：src/content/blog/${DATE}-${SLUG}.md"
echo "📝 编辑完成后，设置 draft: false 并发布"
```

### 自动构建部署脚本

```bash
#!/bin/bash
# deploy.sh

echo "🔨 开始构建..."
pnpm run build

echo "🚀 部署到 VPS..."
scp -r dist/* root@ytaiv.com:/var/www/ytao-ai-observer/

echo "🔄 重启 Caddy..."
ssh root@ytaiv.com "systemctl restart caddy"

echo "✅ 部署完成！"
echo "🌐 访问：https://ytaiv.com"
```
